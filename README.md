# Covid19CasesLookup
App to look up Covid cases from https://covid19api.com/

Uses:
- Jetpack Compose (integration into existing UI)
- BDD style UI tests
- Compose UI tests
- Espresso UI tests
- Unit tests
- MVVM architecture
- StateFlow
- SharedFlow
- Hilt for dependency injection
- View Binding
